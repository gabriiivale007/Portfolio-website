# Portfolio Website

## Description

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS. Showcases projects, skills, and experience with a clean, contemporary design.

## Features

- Fully responsive design
- Modern UI components with shadcn/ui
- Project showcase section
- About and process sections
- Smooth animations with Marquee effects
- Dark mode support
- Mobile-optimized navigation
- Contact information

## Technologies Used

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, PostCSS
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
- **Linting**: ESLint

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Usage

Navigate through the site using the navigation menu to explore projects, learn about the process, and view contact information.

## Project Structure

- `src/components/ui/` - Reusable UI components
- `src/sections/` - Main page sections (Hero, About, SelectedWorks, etc.)
- `src/hooks/` - Custom React hooks
- `src/lib/` - Utility functions
